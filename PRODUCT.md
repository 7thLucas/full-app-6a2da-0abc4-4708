# KYYBOT — Personal AI Assistant

## What it is
KYYBOT is a personal AI assistant web app accessible via Chrome browser. It's built for the owner and their friends to use as a smart, fast, and capable AI chat interface.

## Target Users
- The owner (primary)
- Their friends
- Access method: Chrome web browser (desktop-first, not mobile)

## Core Features (MVP)
1. **AI Chat Interface** — Clean input box + scrollable message thread
2. **General Q&A** — Handles questions on any topic
3. **Coding Help** — Debug code, write code snippets, explain code
4. **Markdown + Code Block Rendering** — Responses with code blocks render cleanly with syntax highlighting
5. **LLM API Integration** — Powered by an AI/LLM API (free/default model; mock cleanly if needed)

## Brand & Tone
- Name: KYYBOT
- Tone: Smart, direct, slightly techy — like a capable dev friend
- Not overly formal; can be casual and helpful
- No fluff, no filler — answers should feel sharp and useful

## Anti-references
- Not a customer support bot
- Not enterprise / corporate
- Not a simple FAQ chatbot

## Strategic Principles
- Keep it fast and responsive
- Prioritize clean output rendering (especially for code)
- MVP-first: ship something usable, then iterate
- Accessible via browser, no install needed
